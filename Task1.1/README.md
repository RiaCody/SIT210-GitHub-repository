This project explores a simple smart assisted-living lighting system built using an Arduino Uno, LEDs, resistors, and a push button. The idea is to simulate a situation where a resident (linda) arrives home and presses a button near the entrance. Instead of needing to manually turn on multiple lights, the system automatically activates lighting that helps guide linda.

In this prototype, two LEDs represent lights in different parts of the home. The first LED represents a porch light at the entrance, and the second represents a hallway light inside the house. When the button is pressed, both lights turn on. After 30 seconds, the porch light turns off while the hallway light stays on for another 30 seconds. 

Although the prototype uses LEDs on a breadboard, it demonstrates how simple embedded systems can support assisted living by improving safety and reducing the need for manual interaction.
